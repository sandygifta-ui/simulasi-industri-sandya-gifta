## 📥 Cara Download Dataset

1. Buka dataset di Kaggle: https://www.kaggle.com/datasets/dinishalikha/dataset-penjualan
2. Download file **dataset_penjualan.csv**.
3. Simpan file ke folder `data/raw/`.
